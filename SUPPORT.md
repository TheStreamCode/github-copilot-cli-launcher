# Support

## Reporting Issues

Open a GitHub issue for bugs and feature requests:

- [GitHub Issues](https://github.com/TheStreamCode/vscode-copilot-cli-launcher/issues)

When reporting a bug, please include:

- VS Code version
- Operating system and version
- Extension version
- Steps to reproduce the issue

## Contact

For direct support enquiries, contact [info@mikesoft.it](mailto:info@mikesoft.it) or visit [https://mikesoft.it](https://mikesoft.it).
